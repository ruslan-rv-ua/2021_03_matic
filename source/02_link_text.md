# "відсутність читабельного тексту в посиланні"

Текст посилання:

- визначається програмно
- описує куди веде посилання
- не містить "посилання на", "перейти до", ...

Як:

- `aria-labelledby`
- `aria-label`
- `<a>...</a>`
- `title` (небажано)
- візуально прихований текст, доступний для асистивних технологій

Добре — візуально прихований текст посилання (Twitter Bootstrap):

	:::html
	<a></a>

## Текст як ID

Погано — два посилання на один документ з різним текстом:

	:::html
	<p><a href="contact.html">Про нас</a></p>
	<p><a href="contact.html">Зв'язатись з нами</a></p>
	
---
Погано: два посилання на різні документи з однаковим текстом:

	:::html
	<p><a href="contact.html"></a>Зв'язатись з нами</a></p>
	<p><a href="directory.html">Зв'язатись з нами</a></p>
